# SlaukoScript Feedback

This repository is for bug reports, feature requests, and discussions about SlaukoScript products.

## Products

- **SlaukoScript Web Dashboard** - Cross-platform community management dashboard at [slaukoscript.com](https://slaukoscript.com)
- **SlaukoBot for Discord** - Moderation, AutoMod, XP/leveling, raid detection, appeals, custom commands, and more
- **SlaukoBot for Twitch** - Full-featured Twitch moderation bot with AutoMod, XP, raid detection, and cross-platform ban sync
- **SlaukoScript Public API** - REST API for external integrations with rate limiting and webhook support

## Features

- Cross-platform moderation (Discord + Twitch)
- Trust-aware AutoMod with spam, link, and caps filtering
- XP/reputation system with leaderboards and role rewards
- Strike escalation with configurable thresholds
- Raid detection with automatic response tiers
- Ban appeal system with SLA tracking
- Cross-platform ban sync between Discord and Twitch
- IFTTT-style automation engine
- Custom commands with permission levels
- Scheduled/timed messages
- Real-time moderation log and live feed
- Mod queue for content review
- Outbound webhooks and API key management
- Stripe-powered billing with subscription tiers
